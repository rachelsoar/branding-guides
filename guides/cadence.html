Rules

Non-negotiables

Every post has a visual

Caption + visual, always. No text-only posts on any platform.

Templates only

Posts are built from approved Canva templates — no freeform design. Grab them from the template library.

When unsure, ask first

Post only what you're confident is on-brand — ask before posting, never after.

Scheduling

Use each platform's native scheduler (free). We'll evaluate Buffer if volume grows.
